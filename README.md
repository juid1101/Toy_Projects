# Toy_Projects
프로그래밍언어 뉴스 요약 프로젝트
# 📰 Daily News Curator: 당신을 위한 뉴스 비서

> **"정보의 홍수 속에서, 딱 필요한 뉴스만 3초 만에 엑셀로 받으세요."**
> 네이버 뉴스의 실시간 데이터를 분야별로 수집하고 정리해주는 자동화 도구입니다.

---

## 1. 💡 기획 의도 및 배경 (Planning)

### "검색도 귀찮은 당신을 위해"
기존의 뉴스 검색 프로그램들은 사용자가 키워드를 직접 입력해야 하는 번거로움이 있었습니다. 또한, "오늘 무슨 뉴스가 중요하지?"라며 막연함을 느끼는 사용자(Cold Start Problem)에게는 검색창이 오히려 장벽이 됩니다.

이 프로젝트는 이러한 문제를 해결하기 위해 **'검색(Search)' 대신 '선택(Selection)'** 방식을 도입했습니다.
사용자는 복잡한 입력 없이 **경제, IT, 사회 등 관심 분야를 클릭 한 번** 하는 것만으로, 지금 당장 알아야 할 헤드라인 뉴스를 요약된 형태로 받아볼 수 있습니다.

**🎯 타겟 사용자:**
* 바쁜 등교 시간, 오늘의 주요 이슈를 빠르게 파악하고 싶은 대학생
* 매일 아침 특정 분야(주식, 부동산 등)의 동향을 스크랩해야 하는 취준생

---

## 2. 🛠 프로그램 기능 및 기술적 특징 (Features & Tech)

이 프로그램은 **[프로그래밍언어] 수업에서 배운 핵심 기술**을 집약하여 구현했습니다.

### (1) 주요 기능
1.  **🔥 실시간 랭킹 대시보드:** 프로그램 시작 즉시 네이버에서 '가장 많이 본 뉴스' Top 5를 크롤링하여 보여줍니다.
2.  **📂 원터치 분야별 큐레이션:** 5대 핵심 분야(경제/IT/사회/정치/세계) 중 하나를 선택하면 해당 섹션의 헤드라인을 수집합니다.
3.  **💾 엑셀(CSV) 자동 아카이빙:** 수집된 뉴스 제목, 요약, 링크를 `Pandas`를 활용해 깔끔한 CSV 파일로 저장하고 다운로드합니다.

### (2) 적용된 강의 내용 (Tech Stack)
* **Web Crawling (Lec 8):** `requests`, `BeautifulSoup`을 활용하여 HTML 구조(DOM)를 파싱하고 필요한 데이터만 정밀하게 추출했습니다.
* **Data Handling (Lec 9):** 수집된 비정형 데이터를 `Pandas DataFrame`으로 구조화하고, `to_csv` 메서드로 영구 저장했습니다.
* **Exception Handling (Lec 7):** 네트워크 불안정이나 HTML 구조 변경 시에도 프로그램이 꺼지지 않도록 `try-except` 구문을 활용해 견고성(Robustness)을 높였습니다.

## 4. 🚀 실행 방법 (How to Run)

이 프로그램은 `Google Colab` 또는 로컬 `Python` 환경에서 실행 가능합니다.

1.  **라이브러리 설치:**
    ```bash
    pip install requests beautifulsoup4 pandas
    ```
2.  **프로그램 실행:**
    ```bash
    python main.py
    ```
3.  **사용법:** 메뉴에서 원하는 번호를 입력하고 엔터(Enter)를 누르세요.
